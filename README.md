# Glow Garden – E-Commerce Website for Skin Care Products  
A full-stack MERN (MongoDB, Express.js, React.js, Node.js) e-commerce platform with secure authentication, product management, order processing, payment integration, and admin dashboard.

---

## Features

### User Features
- Secure registration, login & logout (JWT authentication)
- Search products by keyword or alphabet
- Filter by category and price
- Add to cart, update items, remove items
- Place orders using Braintree payment gateway
- View previous orders and track status
- Update personal profile

### Admin Features
- Manage categories (add, edit, delete)
- Manage products (add, update, delete)
- Manage and update order statuses
- Role-based access (admin-only features)
